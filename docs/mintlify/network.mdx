## Contract addresses (live)

| Contract | Address | Role |
|----------|---------|------|
| `CurvePadFactory` | `0xF54032C714e186bC6e5D84230c3B25cAC2e238Ed` | One-call launch entrypoint |
| `PadRouter` | `0xCA10a8821aF3D54eA9050A279EDd073654f5Fa1C` | The swap desk — every buy/sell goes through it |
| `RewardVault` | `0x5Ca5C1D2D10Bf605F9C42c5Baa0a3f897a3E3811` | Custodies the 0.25% trader/holder [reward](rewards.md) legs; pays capped Merkle claims |
| `FloorCoopFactory` | `0x2615120ECbe93D5DC5e9268337f42817a3224102` | Deploys the per-coin [FloorCoop](floorcoop.md) LP vaults |
| `PlatformFeeSplitter` | `0xF56A82476114BDadC425b850d53FEFCb847e7C65` | Routes the platform's cut (the $ROBIN buy-back split) |
| `WETH` | `0x0Bd7D308f8E1639FAb988df18A8011f41EAcAD73` | Canonical wrapped ETH |
| `UniswapV3Factory` | `0x1f7d7550b1b028f7571e69a784071f0205fd2efa` | The real v3 factory each pool is created on |

The whole stack is **live and source-verified** on the [explorer](https://robinhoodchain.blockscout.com/address/0xCA10a8821aF3D54eA9050A279EDd073654f5Fa1C) (and on [Sourcify](https://sourcify.dev/#/lookup/0xCA10a8821aF3D54eA9050A279EDd073654f5Fa1C) for chain `4663`) — every core contract's source is readable straight from Blockscout, so you can pull any ABI from the explorer or use the typed exports in the [SDK](sdk.md). `CurvePool`, `Bond` and `FloorCoop` addresses are **per-coin**: get `CurvePool`/`Bond` from the factory's `Launched` event or `recordOf(token)` and `router.bondOf(token)`, and a coin's `FloorCoop` from `FloorCoopFactory.coopOf(token)`. Every launched coin's token, curve and bond **auto-verify** on the explorer too, so any coin's source is readable the moment it lands.

The three stateless deployer helpers (reused across launches) are also verified: `LaunchTokenDeployer` `0x55676403eFB000b8667D0F9C6cEdbBF17b9BdcD3`, `CurvePoolDeployer` `0x548Fe951F2022c23bF2e896971aFCD83a39852BB`, `BondDeployer` `0xEe00259A69ab91b9702021571048d1eECbC80eAC`.

## Constants

| Constant | Value | Meaning |
|----------|-------|---------|
| Total supply | `1,000,000,000` | Fixed for every coin (18 decimals) |
| Pool fee tier | `10000` | 1% — the base fee, collected as Uniswap LP fees |
| Base fee | `100` bps | Mandatory 1% per side |
| Max fee per side | `400` bps | A creator may raise a side up to 4% |
| Max opening dev buy | `200` bps | 2% of supply, anti-snipe |
| Graduation reward | `0.5 ETH` | paid to the creator AND the platform at graduation (capped at raise/4 each) |
| Reward leg per side | `25` bps | 0.25% buy → [traders](rewards.md), 0.25% sell → holders (additive) |
| Reward epoch | `7 days` | `finalityDelay` 1d · `challengeWindow` 2d · `claimWindow` 30d |
